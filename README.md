# PRAKTIKUM CHAPTER 2 (CARD WEBSITE)

A card-type website with a cake shop theme called TripleN Bakery. This website has features that are provided to make it easier for users to use.

## Features :
- Navbar 
- Footer 
- Responsive web
- Item search & sorter
